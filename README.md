# Tic Tac Toe AI

A Tic Tac Toe game built in Python with an AI opponent using the Minimax algorithm.

---

## Overview
This project implements a classic Tic Tac Toe game where a human player competes against an AI that plays optimally. The AI evaluates all possible moves using the Minimax algorithm to choose the best outcome.

---

## Features
- Player vs AI gameplay
- AI uses Minimax (optimal decision making)
- Graphical interface built with Pygame
- Restart game functionality

---

## Tech Stack
- Python 3
- Pygame
- NumPy

---

## Installation

1. Clone the repository:
git clone https://github.com/your-username/tic-tac-toe-ai.git
cd tic-tac-toe-ai

2. (Optional) Create and activate a virtual environment:
python3 -m venv env
source env/bin/activate

3. Install dependencies:
pip install pygame numpy

---

## Usage

Run the game:
python TICTACTOE.py

---

## Controls
- Click on a square to place your move
- Press `R` to restart the game
